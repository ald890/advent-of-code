# Advent of code time!

### How to run code:

Requirements: Nodejs 12+ / Bun

1. Install dependencies

```
npm i
```

2. Inside root directory run:

```
npx ts-node 2021/day2
```

or

```
bun 2024/ts/day01/index.ts
```
